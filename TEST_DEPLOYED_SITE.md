## 🔍 문제 해결

### 문제 1: 404 오류 (페이지 새로고침 시)
**해결:**
- Netlify 리다이렉트 설정 확인
- `netlify.toml`과 `public/_redirects` 파일 확인
- 재배포 필요할 수 있음

### 문제 2: API 요청이 localhost로 감
**해결:**
- Netlify 환경 변수 `REACT_APP_API_URL` 확인
- 환경 변수 설정 후 재배포 필요

### 문제 3: 카카오 로그인 실패
**해결:**
- 카카오 개발자 콘솔에서 Redirect URI 확인
- Netlify 환경 변수 `REACT_APP_KAKAO_REDIRECT_URI` 확인
- 카카오 JavaScript 키 확인

### 문제 4: CORS 오류
**해결:**
- 백엔드 CORS 설정 확인
- Netlify 도메인이 백엔드 CORS 허용 목록에 있는지 확인
